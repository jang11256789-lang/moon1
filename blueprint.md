# Lotto Number Generator Project

## Overview
A modern, responsive Lotto Number Generator web application.

## Features
- **Random Number Generation**: Generates 6 unique numbers between 1 and 45.
- **Sorting**: Automatically sorts numbers in ascending order.
- **Animations**: Smooth "pop-in" animations for the generated numbers.
- **Responsive Design**: Works on mobile and desktop devices.
- **Theme Toggle**: Support for both Dark Mode and Light Mode with persistence via LocalStorage.

## Design
- **Theme**: 
  - **Dark Mode (Default)**: Deep navy background with orange/gold accents.
  - **Light Mode**: Clean white/gray background with vibrant orange accents.
- **Typography**: Segoe UI for a modern look.
- **Components**: Floating container with soft shadows and interactive buttons.

## Implementation Details
- **HTML5**: Semantic structure.
- **CSS3**: Uses CSS Variables and Data Attributes for theme switching.
- **JavaScript**: ES6+ logic for number generation and DOM manipulation.
- **Persistence**: Remembers user's theme choice using `localStorage`.
